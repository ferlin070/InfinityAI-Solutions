# 8-Agent AI Command Center
Sistem pengurusan ejen AI pelbagai fungsi yang dikuasakan oleh NVIDIA NIM dan Google Drive.

## Cara Penggunaan
1. Pastikan semua **Secrets** telah diisi di bahagian Settings Hugging Face.
2. Tunggu sehingga status bertukar menjadi **Running**.
3. Dashboard akan muncul secara automatik.
